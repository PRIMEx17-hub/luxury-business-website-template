Luxury Business Website Template

This is a clean, responsive HTML & CSS website template
designed for luxury brands and small businesses.

Files included:
- index.html (main website file)
- style.css (styling)
- assets/ (images and icons)

How to use:
1. Open index.html in a browser
2. Edit text inside index.html
3. Edit colors and styles inside style.css

Built with:
- HTML
- CSS

No frameworks required.
